# PRD: Автосервис Всеволожск - Landing Page

## Дата создания: December 2025

## Описание проекта
Одностраничный лендинг (визитка) для автосервиса во Всеволожске с контактной формой и интеграцией с Яндекс.Картами.

## Пользователи
- Автовладельцы из Всеволожска и Ленинградской области, ищущие СТО для ремонта и обслуживания автомобилей

## Основные требования (Static)
- Фиксированный header с логотипом, телефоном, часами работы
- Hero секция с фоновым изображением и CTA
- Секция услуг (8 карточек)
- Секция "О нас"
- Секция контактов с формой и картой
- Цветовая схема: белый фон, синий #0033A0, красный #DA291C
- Анимации: fade-in при скролле, hover эффекты
- Адаптивность для мобильных устройств

## Реализовано ✅
- Fixed header с навигацией
- Hero секция с параллакс-эффектом
- 8 карточек услуг с hover анимациями
- Секция "О нас" с фото
- Контактная форма (имя, телефон, сообщение)
- Статичная карта со ссылкой на Яндекс.Карты
- Footer с контактами
- Backend API: POST /api/contact
- Полная мобильная адаптивность
- Framer Motion анимации

## Бэклог (P1)
- Интеграция с CRM для заявок
- Email уведомления о новых заявках
- Галерея работ/портфолио

## Бэклог (P2)
- Онлайн калькулятор стоимости услуг
- Отзывы клиентов
- SEO оптимизация (meta tags, sitemap)
